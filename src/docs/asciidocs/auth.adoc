== 인증 Auth

=== 게스트 토큰 발급

---

비회원의 토큰을 발급받을 수 있습니다.

* GET 요청을 보내면 accessToken과 refreshToken을 발급 받을 수 있다,
* 만료일(expiredAt)과 회원 여부(isMember)을 알 수 있다.
* 비회원의 토큰 만료일: 토큰 생성일로부터 1달
